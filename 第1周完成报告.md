## 🚀 快速启动指南

### 前置要求
- Java 11+
- Node.js 16+
- MySQL 8.0
- Maven 3.6+

### 启动步骤

#### Step 1: 初始化数据库
```bash
mysql -h localhost -u root -p123456 < backend/src/main/resources/sql/init.sql
```

#### Step 2: 启动后端服务
```bash
cd backend
mvn spring-boot:run
# 访问：http://localhost:8080
# API文档：http://localhost:8080/doc.html
```

#### Step 3: 启动前端应用
```bash
cd frontend
npm install
npm run dev
# 访问：http://localhost:5173
```

### 访问地址
| 服务 | 地址 |
|-----|------|
| 前端应用 | http://localhost:5173 |
